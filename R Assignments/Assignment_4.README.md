# Assignment4_B573
The purpose fo this assignment was to gain some familiarity with R by creating code to perform some operations on DNA sequences. There are 4 tasks, the first is to load the file in and retrieve the 10th and 758th letter in the sequence. I did this by splicing. The second is to create the reverse compliment of the string and print the 500th through the 800th letters of the sequence. I did this by looping through the sequence to create a new string of the reverse compliment and then printed out the letters. The third task is to create a function that will count the number of times each letter appears in the sequence. I did this by creating empty lists for counting the bases. The fourth and final task is to number of times each nucleotide is contained in the first 1000 base pairs of the sequence. I did this by looping through the kilobases I created and counting each nucleotide base for them.
