The purpose of this code was to do an analysis on data for different genes expressed in normal and tumor tissue. <br>
Part one of the code contains calculations for changing the names of genes in the table, then exploring further by separating normal and tumor data by calculating the expression of the genes. I also calculated the fold change for each row between the tumor and normal data. Lastly I looked into if the genes were expressed higher in the tumor or normal data. <br>
Part two of this code contains visalizations from the analysis prefomed in part one. For the bar graphs in part b and c are the same because after calculating rows with the fold change value more than 5, they all came out to be higher expressed in tumors, so there were none for the normal data. I was then able to create a heatmap to show the expression level of the genes. Lasty I created a cluster map from the same data used for the heatmap. Here I needed to create a smaller subset of the data because when I attempted to use the entire dataset the terminal kept crashing.
